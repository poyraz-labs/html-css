# HTML & CSS Projects

This repository contains a collection of small HTML and CSS practice projects built while learning layout, responsive design, and basic page structure.

## Projects

| Project | Folder | Description |
| --- | --- | --- |
| Amazon Product Page | `amazon-product-page` | A product page inspired by an e-commerce layout. |
| Business Card | `business-card` | A simple personal business card design. |
| Daily Dribble Newsletter | `daily-dribble-newsletter` | A newsletter-style landing page. |
| Card Grid System | `card-grid-system` | A responsive card grid practice project with hover effects. |
| CSS Animation App | `css-animation-app` | A small app focused on CSS animations, motion, and loaders. |
| Feature Section Grid | `feature-section-grid` | A landing page feature section built with grid layout. |
| Gallery Grid System | `gallery-grid-system` | A mixed-size gallery layout using advanced CSS Grid. |
| Google Clone | `google-clone` | A basic clone of the Google home page. |
| Portfolio | `portfolio` | A personal portfolio landing page. |
| Web Agency Hero | `web-agency-hero` | A hero section for a web agency website. |

## Running A Project

Each project folder is a separate Vite app.

```bash
cd project-folder
npm install
npm start
```

## Notes

The projects are focused on practice and learning, so each one keeps the structure simple and easy to understand.
