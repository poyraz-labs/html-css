# Kart Grid Sistemi

Grid tabanlı kart listesi için kısa bir pratik proje. Proje tamamlandı ve responsive kart düzeni, yumuşak hover hareketleri ile birlikte çalışıyor.

## Çalıştırma
```bash
npm install
npm start
```

## Amaç
- Responsive kart grid yapısı kurmak
- Hover efektleri ile küçük hareketler eklemek
- Grid mantığını pratik etmek

## Fazlar
1. İskelet kuruldu.
2. Grid yerleşimi tamamlandı.
3. Görsel stil eklendi.
4. Mikro hareketler eklendi.
5. Kontrol ve not çıkarma yapıldı.
